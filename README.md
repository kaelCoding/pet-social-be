# Pet Social BE

BackEnd for Pet Social with nodejs

## Feature

Similar Facebook but focus on Pet. Features: 

- auth
- profile
- post 
- chat

## Project Setup

```sh
npm install
```

### Migrate Database

```sh
npm run migrate
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Run FrontEnd

[pet-social-fe](https://github.com/cdroot01/pet-social-fe)
